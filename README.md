# Villani Game Factory

A collection of educational games for kids — all self-contained, single-page HTML games that run in any browser with no install or build step required.

**[Play the games here!](https://thomas-villani.github.io/villani-game-factory/)**

## About

These games were built by a dad and his kids with the help of [Claude](https://claude.ai), Anthropic's AI assistant. The whole point was to show the kids that if you can imagine something, you can build it — even if you're just getting started. Each game began as a "what if we made a game where..." conversation and turned into something real they could play and share.

Every game is a single HTML file with no dependencies, no build tools, and no server required. Just open it in a browser and play. This makes them easy to run anywhere — at home, at school, or on a phone.

## Games

### Space Word Catcher
Pilot a spaceship and catch falling letter tiles to spell words before they hit the ground. Covers sight words and common vocabulary with increasing difficulty as you level up.

### Number Blaster
A fast-paced math game where you shoot the correct answer to arithmetic problems. Starts with simple addition and works up through subtraction, multiplication, and division.

### Color Mixer Lab
A virtual color mixing lab where kids drag and combine primary colors to create new ones. Teaches color theory through hands-on experimentation with a fun science-lab theme.

### Map Tap
An interactive geography game — listen to a clue or country name and tap the right location on the map. Covers continents, countries, and landmarks.

### Letter by Letter
A spelling game that reveals a picture as you correctly type each letter of a word. Designed for early readers working on phonics and letter recognition.

### Music Studio
A creative music-making tool where kids can layer beats, melodies, and instrument sounds to compose their own songs. More toy than game — just play and have fun.

### Robo Escape
Guide a robot through puzzle rooms by solving challenges to unlock the next door. Mixes logic puzzles with basic coding concepts like sequences and loops.

### World Explorer
An interactive globe-trotting adventure where kids learn about countries, capitals, flags, and cultures. Includes text-to-speech audio for country names to help with pronunciation.

## Running Locally

Just open any `.html` file in a browser — that's it. No install, no build, no server needed.

To run the full collection with the landing page, you can also use any simple static server:

```sh
npx serve .
```
